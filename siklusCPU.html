Siklus instruksi CPU adalah proses berulang yang dilakukan oleh prosesor untuk menjalankan satu perintah dari program. CPU bekerja dengan sangat cepat, melakukan jutaan siklus setiap detik agar komputer bisa berfungsi dengan baik.

‚öôÔ∏è Tahapan Siklus Instruksi

  • 1. Fetch (Mengambil Perintah): CPU mengambil perintah dari memori untuk diproses.
  • 2. Decode (Menerjemahkan Perintah): CPU memahami makna dari perintah tersebut.
  • 3. Execute (Menjalankan Perintah): CPU menjalankan perintah, seperti menghitung atau memindahkan data.
  • 4. Store (Menyimpan Hasil): Hasil dari proses disimpan ke dalam memori.

  • Execute: CPU menjumlahkan 5 + 3 = 8.
  • Store: CPU menyimpan hasilnya (8) ke variabel A.
